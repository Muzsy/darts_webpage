# ADR-0001 — Aktuális scope leszűkítése

## Döntés

Az első verzió scope-ja kizárólag:

```text
stratégia/kiszállótábla képernyő + score-stratégia adatbázis
```

## Nem része

- gyakorló mód,
- statisztika,
- auth implementáció,
- user profil,
- admin felület,
- AI ajánló.

## Indoklás

A projekt túl gyorsan szétcsúszna, ha már az első körben minden későbbi funkciót beépítenénk.

Először stabil adatmodell és egy jól működő képernyő kell.
